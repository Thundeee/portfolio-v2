Auction House

About

Auction House was my winter 2022 semester assignement. The goal was to deisgn, develop and deploy a website for a client. The client was a local auction house that wanted a website to showcase their services and products. The website was to be responsive and WCAG friendly. I planned and designed the project in FIGMA and used HTML, CSS, Bootstrap and JavaScript to construt the website An API was given to us to use for the project to make calls for creating, updating and deleting information.

I loved working on this assignement and I learned a lot about more advanced JavaScript and CSS tricks which I know will help me in the future.

What I learned

  • Importance of planning when doing bigger projects
  • How to properly set up a repositorys file system and use of modules
  • Proper use of spread syntax in JavaScript
