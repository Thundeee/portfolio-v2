Unite Bot

About

Unite Bot is a project I still work on from time to time. This project uses a bunch of different libraries and tech put together. This is a discord bot for the game "Pokemon Unite". The game does not have a public api but there is a website displaying stats about all playable characters. What this bot does is that it scrapes the website and then converts the data into a JSON file to then display the stats in lists the users want in discord. For example a user can write "!topmon pickrate 10" and the bot would show the top 10 highest pickrate Pokemon the last 7 days. To not "spam" the website, the bot checks if the data is older than 1 hour and if it is, it will scrape the website again and update the JSON file. Another feature on this project that I made was for it to also host an API from the scraped data so that other developers can use the data in their own projects. Since the bot is still currently in development, my API is not public yet! This bot uses Node.js, puppeteer and discord.js.

This project is something I am proud of and I am still working on it from time to time. I have learned a lot from this project and I am still learning new things every time i work on it.

What I learned

  • Web Scraping
  • Proper JSON usage
  • API creation
  • Arguments handling when its either one or multiple
